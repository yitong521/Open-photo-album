3. List the Data of a Specific Movie:

HTTP Status Code Table
Status Code Explanation
500 Internal Server Error
404 Can't find the movie
400 The input ID parameter is invalid
201 List the specific movie

Endpoint: http://localhost:3000/hello/:id (Replace :id with the actual ID)

HTTP Method: GET

Request: Parameters (URL): id, the identifier of the movie

Posted Data (Body): None.

MIME format in response: application/json

Error Response (Status 400):Invalid ID parameter

Example:

input:{"id": a}

output:{"error":"Invalid ID parameter" }

Success Response (Status 200): List a specific movie.

Example:

input:{"id": 1}

output:{ "id":1,

"name":"Arcane",

"year":"2021",

"genre":"animation, action, adventure, tv-show",

"poster":"https://www.nerdpool.it/wp-content/uploads/2021/11/poster-arcane.jpg",

"description":"Set in Utopian Piltover and the oppressed underground of Zaun, the story follows the origins of two iconic League Of Legends champions and the power that will tear them apart." }
